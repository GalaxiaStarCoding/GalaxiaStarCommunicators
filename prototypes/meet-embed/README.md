# Meet-style (Jitsi) embed demo

This demo uses a Jitsi Meet embed (meet.jit.si) inside an iframe. It requires no server and provides a quick Google Meet–like experience.

Open prototypes/meet-embed/client/index.html and change the room name in the UI, or host this project and navigate to that page.

Note: For a production SFU solution, see mediasoup or self-hosted Jitsi instructions. This demo is an easy quickstart.
