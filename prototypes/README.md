# WebRTC Prototypes

This folder contains three small WebRTC prototypes for quick testing and learning:

- teams-1to1: A Teams-styled 1:1 demo (P2P/mesh for two participants) using Socket.IO signaling.
- skype-group: A Skype-styled group mesh demo (small groups, mesh connections).
- meet-embed: A Meet-like demo using a Jitsi Meet iframe for quick server-like experience.

The prototypes share a minimal signaling server located at prototypes/signaling-server.

See each prototype's README for run instructions.
